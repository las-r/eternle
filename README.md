# Eternle
A simple yet extensively customizable Roblox tower guessing game.

All tower info is pulled from Fandom using the [MediaWiki API](https://www.mediawiki.org/wiki/MediaWiki).

*Huge thanks to the members and staff of the EToH Wiki and Discord server for being amazing!*

## Roadmap
- Roblox account authentication
    - For leaderboards and whatnot
- Global stats
    - DAU, etc.
- QOL changes
- More options
    - Realm selection
    - Difficulty selection
