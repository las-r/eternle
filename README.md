# Eternle
A simple yet customizable guessing game about EToH.

All info relating to the game is pulled from the [EToH Wiki](https://jtoh.fandom.com/wiki/) using the [MediaWiki API](https://www.mediawiki.org/wiki/MediaWiki).

*Huge thanks to the members and staff of the EToH Wiki and Discord server for being amazing!*
